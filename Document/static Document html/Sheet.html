ModuleAfter clicking on top right menu button and selecting App settings, a new screen will be displayed with two sections ,
1.General settings: in which we can enable/desable options which are 1. Auto logout, 2. Seat selection, 3. Quick service pop-up forprint bill, 4.Enable kitchen print,
2. Printer setting: here we can enable/disable options which are
A.Bill Printer: here we can select print size 50mm and 80mm and when clicked to enable a pop-up will be displayed with two Tabs 1.bluetooth printers: where all bluetooth devices will be listed if connected via bluetooth, we can select one and click "Set printer" to set it, 2.network printer: here wired printer can be set with printers IP address and port number.
B.Kitchen: same as Bill printer option we select print size 50mm and 80mm and when clicked to enable same pop-up like Bill printer will be displayed to select kitchen prints.		Screenshot

Main screenWhen user is on main screen, This main Screen is divided into 3 partitions,
1.TABLES Part: here at first on left Partition only list of tables will be dislpayed as per the location selected for server login.
after selecting a table, In middle Partition of screen Products will be displayed.
2. Products Part: here all the available products will be displayed with product image, product name and product price, It can be sorted by selecting "groups" option at top which will show all replace the list of products and show list of Menu groups, after selecting a Menu group from list, current list will be replaced with Category list, after selecting a category, list will be replaced with products respective to its category. When a single product is clicked, A pop up will be dislpayed named profuct detail, where we can add modifiers/add-ons and increase the quantity, also shows description of the product on second tab. On this dialog we can click on "Add to Tab" button to add the product. After adding the product the right partition will display cart screen where cart products will be shown.
3. CART Part: here all the products added to tab will be shown with name, price, modifiers/add-ons, add-on's price, quantity increment/decrement, and remove from cart (delete icon). At the bottum of cart Partition, cart ID, Total price and sub menu button (up arrow) is shown, when clicked on up arrow button three options will appear 1. Print: when clicked a print will be created, 2. Done: when clicked kitchen print will be printed, 3. Close: when clicked will open a pop up dialog name payment where Subtotal, Tax,discounts, Total, Paid, Remaining prices and payment method selector will be shown, user can change the payment amount if needed and click Proceed after total amount is paid, it will generate a bill to print.
